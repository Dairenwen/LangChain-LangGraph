# 练习：文档语义拆分

这是一篇专门用来练习语义拆分或结构化拆分的 Markdown 文档。它包含多个主题清晰的小节，适合观察按标题、段落、列表或语义边界拆分时的效果。

## 一、选课前准备

学生在选课前应先阅读专业培养方案，确认本学期需要完成哪些必修课、限选课和任选课。培养方案是判断课程优先级的重要依据，如果没有先确认毕业要求，学生可能会选择很多感兴趣但不计入核心要求的课程。

选课前还应检查个人时间安排。不同课程可能在同一时间段上课，跨校区课程还需要预留通勤时间。对于实验课、体育课和集中实践课，学生应特别关注周次和地点。

## 二、选课过程

进入教务系统后，学生可以按照课程名称、课程编号、教师姓名或课程类别搜索课程。找到目标课程后，应检查课程容量、已选人数、上课时间、考核方式和学分。

如果课程还有余量，并且时间不冲突，学生可以提交选课申请。部分课程可能需要抽签、审核或满足先修条件，因此提交申请并不一定代表最终选课成功。

## 三、退课规则

退课通常只能在学校规定的时间窗口内进行。学生在退课前应确认课程性质，如果课程属于专业必修课，退课可能会影响后续课程修读和毕业进度。

超过退课截止日期后，系统通常不再允许学生自行退课。特殊情况需要提交书面申请，并由学院或教务部门审核。

## 四、常见问题

### 课程人数已满怎么办？

可以等待其他学生退课，也可以选择同类替代课程。如果课程对毕业非常关键，应及时咨询学院教学秘书或辅导员。

### 选课后课表没有显示怎么办？

可能是系统同步延迟，也可能是选课申请尚未审核通过。学生应刷新系统，并查看选课记录中的课程状态。

### 退课后学分不足怎么办？

应尽快补选其他课程，或者咨询学院确认本学期最低学分要求。不要只看单门课程是否退掉，还要看整体学习计划是否仍然合理。

## 五、语义拆分观察点

使用语义拆分时，可以观察拆分器是否把同一主题的内容保留在一起。例如，“退课规则”下面的两段文字应尽量属于同一个主题范围；“常见问题”中的每个问题和对应回答也适合作为独立片段。

如果使用 Markdown 标题拆分器，可以重点观察一级标题、二级标题和三级标题是否被保留到 metadata 中。这样后续做 RAG 检索时，不仅能找到正文，还能知道这段内容属于哪个章节。
